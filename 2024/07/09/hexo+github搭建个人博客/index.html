hexo+github搭建博客

John Doe
2024-07-09

​ 本博客使用的是hexo + github搭建的。网上教程很多，但是也有不少问题，这里简单介绍一下整个安装步骤以及出现问题最多的图片无法显示问题。

环境准备

1. [Git](https://git-scm.com/)
1. [Node.js](https://nodejs.org/)
1. [GitHub账号](https://github.com/)

GitHub

  1. 注册GitHub账号并点击头像个人仓库(Your profile)进入主页，点开仓库(Repositories)新建(New)仓库文件

Hexo

  1. 首先安装Node.js环境，官网下载对应版本直接安装。

  2. 执行命令安装Hexo

  3. 博客本地搭建

  4. 个性化主题，hexo官网模板下载自己喜欢的主题。首先需要安装Git，官网下载对应版本直接安装。git clone克隆主题项目到本地

Hexo生成静态文件部署到github

  1. 修改博客配置文件_config.yml最后部分

​ tips: repo最好改成SSH连接避免部署出错 点开Clone or download，可以选择 SSH 连线、HTTPS 连线。SSH无法使用需要配置，参考链接

​

  1. 要先安装deploy-git，才能用命令部署到GitHub
  1. 生成静态文件并部署到github
  1. 访问github博客http://yourname.github.io

博客文章图片无法显示问题

​ [参考链接](hexo博客中插入图片失败——解决思路及个人最终解决办法_hexo 文章插入图片失败-CSDN博客)

  1. 将_config.yml 文件中的post_asset_folder 选项设为 true 来打开。

  2. 在source\_posts文件夹下新建md文件，以及同名文件夹来存放图片

  3. Typora图片文件的管理，在偏好设置中图像设置。

​ 4.插件下载

​ npm install https://github.com/CodeFalling/hexo-asset-image --save

​ 5.如果还是无法显示删除图片多余前缀，直接写图片名。
